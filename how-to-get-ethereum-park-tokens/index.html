How to Donate or Pay for Parking

To be able to donate to this project and/or pay for parking it is necessary to install the MetaMask blockchain wallet Dapp.

Go to the ‘Add-ons’ or ‘Extensions’ option in your web browser or App Store on your mobile phone or tablet.

Search for ‘MetaMask’

Install the ‘MetaMask’ extension into your browser or mobile phone or tablet.

Once you’ve installed MetaMask go ahead and create a new account protected by a password.

MetaMask will be pre-configured with the Main Ethereum Network and a few Test networks including Ropsten.

In order to use the Binance Smart Chain blockchain network you will need to add the Binance Smart Chain main network and test network to MetaMask. You will also need to add some BNB to the Binance Smart Chain. The following web pages explain how:

Connecting MetaMask to Binance Smart Chain | Binance Academy

Guide: How to add Binance Smart Chain to MetaMask and Send Tokens (crypto-explained.com)

Send BNB Coin to Binance Smart Chain on MetaMask (crypto-explained.com)

Add some ETH or BNB into your MetaMask wallet using the appropriate blockchain network.

Open a browser tab and paste https://parkcryption.azurewebsites.net/ (this will become https://parkcryption.com following the demo stage).

The ParkCryption web app can be installed on your computer or mobile phone or tablet and will work on all devices.

Project donations are appreciated and can be given using the ‘Donate Binance Coin’ and ‘Donate Ethereum’ menu links in the ParkCryption Web and Mobile App.

Choosing ‘Donate Binance Coin’ or ‘Donate Ethereum’ in the ParkCryption App which will open MetaMask and present you with a transaction amount of 0.01 as a donation (you could increase this amount if you want to). Wait for the transaction to be ‘Confirmed’ then please contact us by email with the address used to make the donation and we will send either 10000 BNB PARK tokens (per 0.10 BNB sent) or 10000 ETH PARK tokens (per 0.01 ETH sent).

Once you receive your PARK tokens from us: click ‘Add Token’ in MetaMask and enter the PARK token address provided by us into the ‘Custom Token’ ‘Token Address’, which will display the PARK token then click ‘Next’ to add the PARK tokens to your MetaMask wallet.

Your MetaMask wallet account should now display at least 10000 PARK tokens depending on the amount of your donation.
