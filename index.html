A global vehicle parking solution, using blockchain as the foundation.

ParkCryption is a Web and Mobile application that allows people to find parking spaces anywhere in the world and pay for parking using cryptocurrency as opposed to needing local currency and without the high fees associated with existing platforms.

At the heart of the project is a new cryptocurrency token named ‘PARK’ that has been developed to run on the Binance Smart Chain and Ethereum blockchain networks.

WHY USE PARKCRYPTION

The ParkCryption application is a distributed software application (Dapp) that uses blockchain ledger technology.  Here’s how it works:

  • Customers (anybody that wants to use cryptocurrency as a method of paying to park)
  • Save anywhere between 930% and 1780% on parking costs
  • Instant access to parking places worldwide and the ability to pay using cryptocurrency
  • Customers can view parking profiles before making a decision to reserve a booking
  • Automated parking payments using Blockchain Smart Contracts
  • Parking Space Providers (anybody that wants to rent out a parking space and receive cryptocurrency as a method of payment)
  • Keep anywhere between 930% and 1780% of your parking rental income
  • Instant access to people wanting to rent a parking space worldwide
  • Instant payment without risk of chargebacks
  • No unjust negative reviews as all transactions can be proved

SECURITY AND PRIVACY

ParkCryption will use two factor authentication (2FA) to provide an extra layer of security beyond passwords and usernames.  2FA works by creating and sending information to the user or parking provider’s mobile phone to verify their identity at the point of payment and during user registration.

FREE TO USE

ParkCryption is free to use and payment transaction fees are almost zero for parking anywhere in the world unlike competitors who charge high commissions for using their Apps and card transaction fees as well as cross-border currency conversion fees when parking in a country where you do not have a credit card.

GLOBAL CURRENCY

The ParkCryption token is based on cryptocurrency/blockchain technology, allowing users to provide and use parking services without the restrictions of bank fees and fiat currencies.

HIGH SECURITY

Payments are approved using Two Factor Authentication (2FA).  This additional level of security ensures transactions are secure.

BLOCKCHAIN SMART CONTRACTS

ParkCryption uses Blockchain Smart Contracts to make payment only when the user’s vehicle has been recorded using satellite coordinates as been parked at a designated parking spot for the reserved amount of time, which protects users and parking providers from fraud.

Mobile Distributed Application

The ParkCryption App uses a decentralized blockchain platform and payment ledger to ensure secure commerce and protection from unathorised third-party interference. 

DATA SCIENCE SYSTEM AND ALGORITHMS

Data science algorithms will recommend suitable parking places to customers in real-time as they drive into a town or city.

January 2020

Project Idea and Planning Stage

Project planning stage and idea documentation.

March 2020

Technical Analysis and High Level Application Design

Carried out technical analysis of the Ethereum blockchain functions (Binance Smart Chain is a fork of Ethereum) and the available software development tools and software languages to use to interact with each blockchain, write smart contracts etc.  Began Ethereum Solidity Smart Contract development and testing along with ParkCryption software application design at a high level including which software development languages and RAD tools to use for the project.

July 2020

Begin ParkCryption application development

Software development work starts on the ParkCryption software system using C# .Net with the Blazor Web Framework along with JavaScript Interop, Azure Cloud Hosting, SQL Server, Entity Framework Core, Ethereum Solidity and Bootstrap 4.0.  C# and JavaScript were used to interact with the MetaMask wallet.  C# and Solidity Smart Contracts where used to interact with the Binance Smart Chain and Ethereum blockchains for PARK token creation and parking space reservation functionality.

March 2021

Minimal Viable Product (MVP) release
